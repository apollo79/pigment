# @kobalte/pigment-tailwind-preset

Pigment preset for Tailwind CSS

## Installation

```bash
npm install -D @kobalte/pigment-tailwind-preset
# or
yarn add -D @kobalte/pigment-tailwind-preset
# or
pnpm add -D @kobalte/pigment-tailwind-preset
```
