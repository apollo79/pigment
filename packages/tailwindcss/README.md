# @kobalte/pigment-tailwindcss

Pigment preset for TailwindCSS.

## Installation

```bash
npm install -D @kobalte/pigment-tailwindcss
# or
yarn add -D @kobalte/pigment-tailwindcss
# or
pnpm add -D @kobalte/pigment-tailwindcss
```
