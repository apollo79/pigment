# @kobalte/pigment-tailwind-plugin

Pigment plugin for Tailwind CSS.

## Installation

```bash
npm install -D @kobalte/pigment-tailwind-plugin
# or
yarn add -D @kobalte/pigment-tailwind-plugin
# or
pnpm add -D @kobalte/pigment-tailwind-plugin
```
