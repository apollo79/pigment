# Server side rendering

## Usage with SolidStart

If you are using [SolidStart](https://start.solidjs.com/) you will need to add the ssr setting to your `vite.config.ts`:

```ts {5-7}
import solid from "solid-start/vite";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [solid()],
  ssr: {
    noExternal: ["@kobalte/pigment"],
  },
});
```

<Alert color="warning" variant="soft">
  <span>Pigment has been tested with `solid-js@1.7.3` and `solid-start@0.2.26`, </span>
  <span>compatibility with other versions is not guaranteed.</span>
</Alert>
