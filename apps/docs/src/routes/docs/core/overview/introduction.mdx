# Introduction

Pigment is a full-featured UI library for SolidJS. It provides ready to use components for building **web applications** with a consistent look and feel.

## Key features

### Simple

Designed to expose a minimalist and simple API so that you can quickly build want you want, avoiding having to import multiple components to display just one.

### Accessible

Built on top of [Kobalte](https://kobalte.dev) a UI toolkit for building accessible web apps and design systems with SolidJS..

### Consistent

Comes with a default theme with support for light and dark modes that ensures a consistent experience in your product.

### Themeable

Provides a large set of customizable design tokens to make it look the way you want.
