# Introduction

<Alert status="warning">
  <span>Pigment is not available yet on `npm`.</span>
</Alert>

Pigment is a full-featured UI library for SolidJS built with [Kobalte](https://kobalte.dev) and [Tailwind CSS](https://tailwindcss.com/). It provides ready to use components for building **web applications** with a consistent look and feel.

## Principles

### Simple

Designed to expose a minimalist API so that you can quickly build what you want.

### Accessible

Built on top of [Kobalte](https://kobalte.dev) a UI toolkit for building accessible web apps and design systems with SolidJS.

### Consistent

Comes with predefined themes with support for light and dark modes that ensures a consistent experience in your application.
