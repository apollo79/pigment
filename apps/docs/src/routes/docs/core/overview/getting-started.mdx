# Getting started

Learn how to install and start using Pigment.

## Adding the Tailwind CSS preset

Make sure that you have [Tailwind CSS](https://tailwindcss.com/docs/guides/solidjs) installed, then inside the `tailwind.config.cjs` file:

1. Add Pigment in the `content` array, in order to get the classes applied to the components.
2. Require Pigment as a preset.

```js {2,6}
module.exports = {
  content: [
    "./node_modules/@kobalte/pigment/dist/**/*.{js,cjs,mjs}",
    "./src/**/*.{html,js,jsx,ts,tsx,mdx}",
  ],
  theme: {},
  presets: [require("@kobalte/pigment-tailwindcss")],
};
```

<Alert status="warning">
  <span>Pigment is designed to work with Tailwind's default theme, without prefix, </span>
  <span>and with [preflight](https://tailwindcss.com/docs/preflight) enabled.</span>
</Alert>
