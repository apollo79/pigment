# Alert

Display a brief, important message in a way that attracts the user's attention without interrupting the user's task.

## Import

```tsx
import { Alert } from "@kobalte/pigment";
```

## Usage

### Variants

The Alert component supports two Pigment's global variants: `solid` and `soft` (default).

<Preview isCentered={false}>
  <WithVariants />
</Preview>

```tsx
<div class="flex flex-col space-y-4">
  <Alert variant="solid" color="info" title="Software update">
    A new software update is available.
  </Alert>
  <Alert variant="soft" color="info" title="Software update">
    A new software update is available.
  </Alert>
</div>
```

### Colors

The Alert component supports six different color which affect the color scheme and icon used: `primary` (default), `neutral`, `success`, `info`, `warning` and `danger`.

<Preview isCentered={false}>
  <WithStatuses />
</Preview>

```tsx
<div class="flex flex-col space-y-4">
  <Alert variant="soft" color="neutral" title="Service temporarily unavailable.">
    We are currently migrating our infrastructure.
  </Alert>
  <Alert variant="soft" color="success" title="Well done.">
    The data has been successfully saved on our server.
  </Alert>
  <Alert variant="soft" color="info" title="Software update.">
    A new software update is available.
  </Alert>
  <Alert variant="soft" color="warning" title="Attention needed.">
    Your registration token is about to expire.
  </Alert>
  <Alert variant="soft" color="danger" title="Something went wrong.">
    There was an error processing your request.
  </Alert>
</div>
```

### Custom icon

The `icon` prop is used to change the Alert icon. It accepts either a `JSX.Element` or a function receiving the Alert's color and should return a `JSX.Element`.

<Preview isCentered={false}>
  <WithCustomIcon />
</Preview>

```tsx
<Alert variant="soft" color="info" title="Software update" icon={<RocketIcon />}>
  A new software update is available.
</Alert>
```

### Dismissible

Use the `dismissible` prop paired with the `onDismiss` event handler to make a dismissible alert. Additionally, the `dismissButtonLabel` prop can be used to provide an accessible label for the dismiss button.

<Preview isCentered={false}>
  <WithDismissButton />
</Preview>

```tsx
<Alert
  variant="soft"
  color="info"
  title="Software update"
  dismissible
  dismissButtonLabel="Close"
  onDismiss={() => alert("dismissed")}
>
  A new software update is available.
</Alert>
```

### Multiline

The Alert component display it's content on a single line by default. Use the `multiline` prop to display a multiline alert.

<Preview isCentered={false}>
  <WithMultiline />
</Preview>

```tsx
<Alert variant="soft" color="info" title="Software update" multiline>
  A new software update is available.
</Alert>
```

## API Reference

### Component identifier

The name `Alert` can be used when providing [default props](/docs/core/customization/themed-components#default-props) or [slot classes](/docs/core/customization/themed-components#slot-classes) in the theme.

### Props

| Prop               | Description                                                                                                                                                                                                                                                                    |
| :----------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| variant            | `"solid" \| "soft"` <br/> **default:** `"soft"` <br/> The visual style of the alert.                                                                                                                                                                                           |
| color              | `"neutral" \| "success" \| "info" \| "warning" \| "danger"` <br/> **default:** `"info"` <br/> The color of the alert, which affect the color scheme and icon used                                                                                                              |
| dismissible        | `boolean` <br/> **default:** `false` <br/> Whether the alert should show a dismiss button.                                                                                                                                                                                     |
| multiline          | `boolean` <br/> **default:** `false` <br/> Whether the alert title and description should appear on multiple lines.                                                                                                                                                            |
| title              | `JSX.Element` <br/> The title of the alert.                                                                                                                                                                                                                                    |
| children           | `JSX.Element` <br/> The description of the alert.                                                                                                                                                                                                                              |
| icon               | `((color: "neutral" \| "success" \| "info" \| "warning" \| "danger") => JSX.Element) \| JSX.Element` <br/> The icon displayed next to the title. Can be a function receiving the alert color, useful when defining all color icons in the Pigment theme.                       |
| dismissButtonLabel | `string` <br/> **default:** `"Dismiss"` <br/> An accessible label for the dismiss button. The default label is internationalized, you can check the list of supported languages [here](https://github.com/kobaltedev/kobalte/blob/main/packages/core/src/i18n/common.intl.ts). |
| onDismiss          | `() => void` <br/> Event handler called when the dismiss button is clicked.                                                                                                                                                                                                    |

### Slots

| Name          | CSS selector               | Description                 |
| :------------ | :------------------------- | --------------------------- |
| root          | `.pg-alert-root`           | Root element                |
| icon          | `.pg-alert-icon`           | Icon wrapper                |
| content       | `.pg-alert-content`        | Wraps title and description |
| title         | `.pg-alert-title`          | Title                       |
| description   | `.pg-alert-description`    | Description                 |
| dismissButton | `.pg-alert-dismiss-button` | Dismiss button              |
