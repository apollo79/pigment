# Global Variants

Pigment uses the concept of global variants inspired by [Joy UI](https://mui.com/joy-ui/main-features/global-variants/) which aim to improve consistency across every component in the library.

## Main variants

Pigment provides four main variants: `solid`, `soft`, `outlined`, and `ghost`. These variants are intended to cover the majority of use cases in modern web design.

Each variant conveys a different level of importance in the user interface:

- `solid` is best suited for primary elements and the most important actions on the page.
- `soft`, `outlined`, and `ghost` are better for secondary and tertiary actions.

<Preview isRounded>
  <div class="flex items-center gap-2.5 flex-wrap">
    <Button variant="solid">Solid</Button>
    <Button variant="soft">Soft</Button>
    <Button variant="outlined">Outlined</Button>
    <Button variant="ghost">Ghost</Button>
  </div>
</Preview>

## Input variants

In addition to the main variants, the `soft` and `outlined` input variants are designed specifically for form UI elements such as text fields, select, checkboxes and radio buttons.

<Preview isRounded>
  <div class="grid grid-cols-1 sm:grid-cols-2 gap-2.5">
    <TextField variant="soft" placeholder="TextField" />
    <TextField variant="outlined" placeholder="TextField" />
    <Select
      variant="soft"
      options={["Apple", "Banana", "Blueberry", "Grapes", "Pineapple"]}
      placeholder="Select"
    />
    <Select
      variant="outlined"
      options={["Apple", "Banana", "Blueberry", "Grapes", "Pineapple"]}
      placeholder="Select"
    />
    <Checkbox variant="soft" label="Checkbox" />
    <Checkbox variant="outlined" label="Checkbox" />
  </div>
</Preview>

## Selection variants

Pigment's `solid` and `soft` selection variants are intended for elements in a selected or checked state, such as select options, menu items, checkboxes, switches and radio buttons.

<Preview isRounded>
  <div class="space-y-2.5">
    <Select
      class="w-[177px]"
      defaultValue="Apple"
      options={["Apple", "Banana", "Blueberry", "Grapes", "Pineapple"]}
      placeholder="Select"
    />
    <Checkbox label="Checkbox" defaultIsChecked class="self-center" />
  </div>
</Preview>
