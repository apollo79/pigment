- `maroon` is based on desaturated red hues.

## Composing a color palette

### Neutral pairing

`gray` work well with any hue if you want a neutral vibe.

### Natural pairing

Choose the gray scale which is saturated with the hue closest to your brand color to create a more harmonious vibe.

### Choosing semantic colors

Pigment uses the concept of colors to communicate critical information, a favorable outcome or caution.

Here are some common pairings you can do in your theme with the raw colors described above and Pigment semantic colors:

- **success:** lime/green/emerald/teal
- **info:** cyan/sky/blue/indigo
- **warning:** orange/amber/yellow
- **danger:** pink/rose/red
- **discovery:** violet/purple/fuchsia
